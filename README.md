# Smart College Digital Twin

## 📖 Project Overview

The Smart College Digital Twin is a simulation-based IoT project developed using ESP32 and Wokwi. The system monitors various environmental parameters of a college campus in real time. Sensor data is transmitted using MQTT, processed through Node-RED, stored in InfluxDB, and visualized using Grafana dashboards.

---

## 🚀 Features

- Real-time environmental monitoring
- MQTT-based wireless communication
- Data processing using Node-RED
- Time-series data storage with InfluxDB
- Interactive Grafana dashboard
- Simulation using Wokwi
